# Learn Event-Driven Architecture

**Interactive learning website designed for ADHD learners**

Master Event-Driven Architecture through bite-sized lessons, hands-on coding, and gamified progress tracking.

---

## 🎯 Features

### ADHD-Optimized Learning
✅ **Bite-sized lessons** - 5-15 minute focused sessions (Pomodoro-friendly)
✅ **Immediate feedback** - XP rewards for completing lessons
✅ **Visual progress tracking** - See your advancement clearly
✅ **Gamification** - XP system, streaks, achievements
✅ **Interactive examples** - Learn by doing, not just reading
✅ **Mobile-first design** - Learn anywhere, anytime

### Course Structure

**Module 1: Foundations (5 lessons)**
- What is Event-Driven Architecture?
- Traditional vs Event-Driven comparison
- Core components (Events, Publishers, Subscribers)
- Build your first event flow
- Foundations quiz

**Module 2: EDA Patterns (6 lessons)**
- Pub/Sub Pattern
- Event Sourcing
- CQRS (Command Query Responsibility Segregation)
- Event Streaming
- Choreography vs Orchestration
- Pattern recognition quiz

**Module 3: Build Real Systems (5 lessons)**
- Node.js project setup
- In-memory EventEmitter system
- Redis Pub/Sub implementation
- Event Sourcing with Event Store
- Final project: Complete notification system

**Module 4: Production Ready (4 lessons)**
- Error handling & dead letter queues
- Monitoring & observability
- Scaling event-driven systems
- Final assessment

---

## 🚀 What You'll Build

By the end of this course, you'll have built:

1. **In-Memory Event System** - Basic EventEmitter pattern
2. **Redis Pub/Sub System** - Multi-process event communication
3. **Event Sourcing App** - Event store and event replay
4. **Complete Notification System** - Production-ready event-driven application

**1. Bite-Sized Content**

- No long walls of text

**2. Active Learning**
- Interactive code examples

**3. Immediate Rewards**
- XP for completed lessons
- Visual progress bars

**4. Clear Structure**
- Numbered lessons in logical order
